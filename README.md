# Covid-Worldwide-Data-Website


# Description:
It shows the COVID-19 cases all over the world country wise.

# Create Project:
django-admin startproject covid   

# Create the app:
python3 manage.py startapp covidapp

# Working: 
Select Country and click on submit button.
